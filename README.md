# The Campfire Stories

**Five-minute bedtime stories for ages 3–7, built from cross-cultural story arcs.**

Each book contains two paired stories that explore the same theme from two different storytelling traditions — Akan spider tales, Buddhist Jātaka stories, Aboriginal Dreamtime narratives, Haudenosaunee gratitude traditions, Japanese folk tales, and more.

These stories are designed to be read aloud. They include darkness — because darkness with a container is how children build courage. They include refrains — because children need to participate, not just listen. They include warmth — because every scary story should end in an embrace.

## The Books

| Book | Theme | Traditions |
|------|-------|------------|
| **Book 1: When Things Feel Scary** | Courage and fear | Akan (Anansi) + Jātaka (Buddhist) |
| **Book 2: When Someone Needs Help** | Compassion and sacrifice | Jātaka + Haudenosaunee |
| **Book 3: When the World Feels Too Big** | Finding your place | Aboriginal Australian + Japanese |
| **Book 4: When Things Aren't Fair** | Cleverness and justice | Akan (Anansi) + 1001 Nights |
| **Book 5: When You Feel Different** | Belonging and gifts | Grimm/European + Haudenosaunee |

## Methodology

Every story passes the **Three-Filter Test**:

1. **Useful** — Builds a specific developmental skill
2. **Fits the data** — Developmentally appropriate, honors the source tradition
3. **Compassionate** — Serves the child's growth, darkness has a container

## How This Was Made

This book was written using [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview) and the methodology described in [Vibe Writing 102](https://recursive.eco).
